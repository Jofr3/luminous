# Pokemon TCG Knowledge Base

This directory contains a comprehensive reference for the rules and mechanics of the Pokemon Trading Card Game (TCG). It is compiled from official sources including the official rulebook (par_rulebook_en.pdf), the quick start rules, the tournament rules handbook, the official errata list, and the Pokemon Rulings Compendium.
